# 🎓 Student Performance Analysis & Prediction

## 📌 Project Overview

This project focuses on analyzing student performance data and building a Machine Learning model to predict student scores based on various academic and demographic factors.

The goal is to understand patterns in student performance and apply basic regression techniques to make predictions.

---

## 🧠 Key Features

* 📊 Exploratory Data Analysis (EDA) using visualizations
* 🧹 Data preprocessing and cleaning
* 🏗️ Feature engineering (Total Score, Average Score)
* 🤖 Model building using Linear Regression
* 📈 Model evaluation using performance metrics

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## 📊 Model Performance

The model was evaluated using standard regression metrics:

* R² Score: **~0.15**
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

📌 **Note:**
The R² score indicates that the model explains a limited portion of variance in the dataset. This highlights the need for more advanced modeling techniques and better feature engineering.
